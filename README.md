# Apple Glass UI Bottom Menu

A stylish, interactive bottom menu dock inspired by macOS/iOS glassmorphism design, built with HTML, CSS, and GSAP.

## Features

- **Glassmorphism UI**: Frosted glass effect with blur and semi-transparent background.
- **Interactive Animations**: Smooth hover effects powered by GSAP.
- **Responsive**: Centers perfectly at the bottom of the screen.
- **Full Screen Background**: Immersive background image experience.

## Technologies Used

- HTML5
- CSS3 (Backdrop Filter, Flexbox)
- JavaScript
- [GSAP](https://greensock.com/gsap/) (GreenSock Animation Platform)

## How to Run

1. Clone the repository.
2. Open `index.html` in your web browser.
